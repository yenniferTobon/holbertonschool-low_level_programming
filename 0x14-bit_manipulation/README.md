### 0x14. C - Bit manipulation

### DESCRIPTION

Bit manipulation is the act of algorithmically manipulating bits or other pieces of data shorter than a byte. 
C language is very efficient in manipulating bits.

### EXAMPLES

A = 0011 1100
B = 0000 1101

-----------------

A&B = 0000 1000
A|B = 0011 1101
A^B = 0011 0001
~A  = 1100 0011
